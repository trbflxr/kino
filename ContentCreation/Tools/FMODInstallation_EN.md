# Installing FMOD Studio

CarX currently uses **FMOD Studio version `2.02.00`**. That’s the exact version you need to install.

## Downloading FMOD Studio

Go to the [FMOD Studio download page](https://www.fmod.com/download#fmodstudio) and download version `2.02.00` for your platform, then install it.

![sounds_fmod_download.png](../Images/Sounds/sounds_fmod_download.png)

There’s nothing special about the installation process — just install it like any other application.
